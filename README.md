# Solar System Sizer PWA

A professional Progressive Web App for sizing solar energy systems based on the EEF672: Hybrid Renewable Energy Systems course.

## Features
- Complete solar system sizing algorithm
- Load calculator with appliance database
- Battery bank sizing with type comparison
- PV array sizing with panel database
- Cost estimation and ROI calculator
- PWA features: Installable, works offline
- Dark/light theme
- Export reports
- GPS-based peak sun hour estimation

## Live Demo
[View Live](https://KewirBinnyuy.github.io/solar-system-sizer/)

## How to Use
1. Add your appliances
2. Configure system parameters
3. View battery and PV sizing
4. Check cost estimation
5. Install as PWA for offline use

## Technologies
- HTML5
- CSS3 (CSS Grid, Flexbox)
- JavaScript (ES6+)
- Service Workers
- PWA Manifest

## Deployment
This app is deployed using GitHub Pages.
